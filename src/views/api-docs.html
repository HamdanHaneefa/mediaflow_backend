üîê Authentication

The API uses JWT (JSON Web Tokens) for authentication. Include the token in the Authorization header for all requests.

POST /auth/register
Register a new user account

Request Body required

Parameter Type Description
email string User email address
password string Password (min 6 characters)
firstName string User's first name
lastName string User's last name

Example Request

{ "email": "user@example.com", "password": "SecurePass123", "firstName": "John", "lastName": "Doe" }

Example Response (200)

{ "success": true, "message": "User registered successfully", "data": { "accessToken": "eyJhbGc...", "refreshToken": "eyJhbGc...", "user": { "id": "uuid", "email": "user@example.com", "firstName": "John", "lastName": "Doe" } } }
POST /auth/login
Login with email and password

Request Body

{ "email": "user@example.com", "password": "SecurePass123" }
POST /auth/refresh
Refresh access token
GET /auth/me
Get current authenticated user details
POST /auth/logout
Logout and invalidate tokens
POST /auth/forgot-password
Request password reset email
POST /auth/reset-password
Reset password with token
PATCH /auth/change-password
Change current password

üìá Contacts Management

Manage clients, vendors, freelancers, and partners.

POST /contacts
Create a new contact

Request Body

Parameter Type Description
name required string Contact name
email required string Email address (unique)
phone string Phone number
company string Company name
role enum Client | Vendor | Freelancer | Partner
status enum Active | Inactive | Pending
tags array Tags for categorization

Example Request

{ "name": "Acme Productions", "email": "contact@acme.com", "phone": "+1-555-0100", "company": "Acme Productions Inc", "role": "Client", "status": "Active", "tags": ["vip", "corporate-video"] }
GET /contacts
List all contacts with pagination and filters

Query Parameters

Parameter Type Description
page number Page number (default: 1)
limit number Items per page (default: 10)
search string Search by name, email, or company
role enum Filter by role
status enum Filter by status
sortBy string Sort field (default: created_at)
sortOrder enum asc | desc (default: desc)
GET /contacts/:id
Get contact details with relationships
GET /contacts/:id/stats
Get contact statistics (projects, tasks, revenue)
PUT /contacts/:id
Update contact information
DELETE /contacts/:id
Delete a contact (checks for dependencies)

üìÅ File Upload

Upload images, documents, videos, and other files with type-specific validation.

POST /upload/image
Upload a single image (10MB limit)

Supported Formats

jpeg, jpg, png, gif, webp

Form Data

Parameter Type Description
file required file Image file to upload
POST /upload/images
Upload multiple images (max 10 files)
POST /upload/document
Upload a single document (50MB limit)

Supported Formats

pdf, doc, docx, xls, xlsx, ppt, pptx, txt, csv

POST /upload/documents
Upload multiple documents (max 10 files)
POST /upload/video
Upload a video file (500MB limit)

Supported Formats

mp4, avi, mov, wmv, flv, mkv, webm

POST /upload/avatar
Upload an avatar image (5MB limit)
POST /upload/receipt
Upload a receipt image (10MB limit)
POST /upload/any
Upload any file type (100MB limit)
